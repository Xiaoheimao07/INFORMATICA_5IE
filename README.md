# TalkMusic

Piattaforma web per la pubblicazione e la ricerca di strumenti musicali usati, con sistema di recensioni e follow tra utenti.

## Funzionalità
- Registrazione e login utenti
- Pubblicazione e eliminazione di articoli 
- Sistema di follow tra utenti
- Recensioni su  articoli

## Struttura 
```
TalkMusic/
├── index.php               
├── database.sql            
├── css/                    
├── Immagini/

    ├── include/

```
